# Dashboard de Monitoramento: Mpox Brasil 2026
<img width="1024" height="1024" alt="6dd4305e-b1ee-4953-8335-a771184a8bf7" src="https://github.com/user-attachments/assets/f59cce37-8790-400c-bef8-465484795990" />


Criei este projeto para transformar dados complexos em um visual fácil de entender. É um dashboard interativo que mostra a situação da Mpox no Brasil agora em 2026.

### O que ele faz:
- Mostra o total de casos confirmados e suspeitos no país.
- Permite escolher um estado para ver os números específicos de lá.
- Gera um gráfico automático que compara os estados mais atingidos.

### O que usei:
- **Python**: A base de tudo.
- **Streamlit**: Para criar a interface do site e os filtros.
- **Pandas**: Para organizar a tabela de dados.
- **Plotly**: Para desenhar o gráfico interativo.

### 🚀 Como testar:
Com o Python instalado, é só rodar no terminal:
1. `pip install streamlit pandas plotly`
2. `streamlit run app.py`
